Jogo 2D com mecânicas

Projeto desenvolvido na Unity com foco em estudo de mecânicas de jogos 2D.

O objetivo deste projeto foi praticar conceitos de desenvolvimento de jogos, movimentação de personagem, interação com o ambiente e lógica de gameplay utilizando a Unity.

Tecnologias utilizadas

Unity

C#

Mecânicas de jogos 2D

Estrutura do projeto

Assets → scripts, sprites e recursos do jogo
Packages → dependências do projeto
ProjectSettings → configurações da Unity

Objetivo do projeto

Este projeto foi desenvolvido como estudo prático de desenvolvimento de jogos e implementação de mecânicas básicas dentro da Unity sempre se divertindo com a programação, com tiles e sprites totalmente autorais.
